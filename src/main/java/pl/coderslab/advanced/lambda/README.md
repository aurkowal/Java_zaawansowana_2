1. Napisz program, który:
- pobierze od użytkownika dowolną liczbę słów 
- umieści je w liście
- posortuje alfabetycznie w kolejności Z-A ignorując wielkość znaków

Do sortowania kolekcji należy użyć wyrażenia lambda.


## Zadanie 2 - rozwiązywane z wykładowcą

Korzystając z interfejsu funkcyjnego **Predicate** napisz wyrażenie lambda, które będzie sprawdzało,
czy podany parametr jest typu liczbowego (**Number**).

Zweryfikuj działanie w następujący sposób:

```java
boolean test1 = predicate.test(123);
System.out.println(test1);

boolean test2 = predicate.test(4f);
System.out.println(test2);

boolean test3 = predicate.test("seven");
System.out.println(test3);
```
